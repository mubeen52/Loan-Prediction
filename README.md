
# 🏦 SmartLoan AI | Automated Eligibility Engine

A high-end Fintech web application that leverages Machine Learning to predict loan approval probabilities. This project features a premium "Cyber-Glassmorphism" interface and a robust Scikit-Learn backend.

## 🌟 Features
*   **AI-Powered Predictions:** Uses a Random Forest Classifier for high-accuracy decision making.
*   **Fintech UI:** Modern, responsive dashboard with neon accents and real-time visual feedback.
*   **Automated Pipeline:** End-to-end workflow from raw data processing to web deployment.
*   **Dynamic Feedback:** Real-time color-coded alerts (Green for Approved, Red for Rejected).
